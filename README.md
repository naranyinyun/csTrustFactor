# Intro

此项目旨在为玩家提供其他玩家（或者他们自己）相关的 CS 游戏信息、信誉因子（非官方，自行评估）、信誉得分、置信度以及可能的 AI 参考建议。

它的功能可能类似于 trustfactor.gg，但我们使用的评分标准与他们完全不同。作为开源项目，我们公开所有源代码和实际使用的评分标准，也欢迎各位为项目出一份力，这最终有利于所有用户。

各项功能正在逐步完善，评分标准正在设计，此项目目前属于活跃开发期。

## 收集的信息以及处理方式

我们从 Steam Web API 收集公开可获得的玩家数据，这些数据包括：

- 任何类型的封禁、交易限制和账户限制
- 账户使用年限
- CS 游戏时长
- 以及其它未来添加的数据

## 评分标准

信誉得分采用扣分制，满分为 100，以下是评分细则：

### 游戏封禁

包括 VAC 封禁、游戏封禁、开发者封禁和交易限制。

这些封禁不计次数、时长、游戏类型和原因。

这些类型计 0 分，不再计算其他得分。

### 账户年限

根据账户年限的时长，分为：

- 小于一年：扣除 30 分
- 一至三年：扣除 10 分
- 三至六年：不计分
- 大于六年：加 10 分

### 游戏时长

只计算 CS

- 小于 50 小时：扣除 60 分
- 50 至 300 小时：扣除 20 分
- 300 至 1000 小时：不计分
- 大于 1000 小时：加 10 分

### 账户使用情况

距离上次使用时长：

- 大于 8765 小时：扣除 50 分
- 730 至 8765 小时：扣除 30 分
- 0 至 730 小时：不计分

### 资料不公开

未设置个人资料、个人资料不公开、游戏数据不公开以及其它无法查询资料的，计 0 分，不再计算其他得分，置信度计 0 分（结果不可信）。
